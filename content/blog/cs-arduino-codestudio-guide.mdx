## Зачем учителю эта инструкция

Подключение Arduino к компьютеру через классическую Arduino IDE — процесс, который требует установки драйверов, выбора правильного порта, настройки платы. Для учителя, который не является профессиональным программистом, это может быть стрессовым опытом.

CodeStudio упрощает этот процесс до минимума. Но даже простой процесс требует понимания. В этой статье мы дадим **пошаговую инструкцию**, которая позволит любому учителю подключить Arduino к CodeStudio за 10 минут.

## Что вам понадобится

### Оборудование

- **Arduino Uno** (рекомендуется для начала) или Arduino Mega, Nano
- **USB-кабель** типа A-B (обычно идёт в комплекте с платой)
- **Компьютер** с браузером Chrome, Edge или Firefox
- **Стабильное интернет-соединение**

### Программная часть

- **Аккаунт учителя в CodeStudio** (бесплатная регистрация на сайте)
- **CodeStudio Agent** — небольшая программа, которая обеспечивает связь браузера с USB-портом

## Шаг 1: Установка CodeStudio Agent

CodeStudio Agent — это лёгкая программа (менее 20 МБ), которая работает в фоновом режиме и обеспечивает связь между браузером и USB-портами компьютера. Без неё браузер не может напрямую общаться с Arduino — это ограничение безопасности всех современных браузеров.

**Процесс установки:**

1. Войдите в свой аккаунт CodeStudio
2. Перейдите в раздел «Настройки» → «Подключение устройств»
3. Нажмите «Скачать Agent» — система автоматически определит вашу ОС
4. Запустите скачанный файл и следуйте инструкциям установщика
5. После установки в системном трее появится иконка CodeStudio Agent

**Важно:** Agent устанавливается один раз на компьютер. На школьных компьютерах попросите системного администратора установить Agent на все машины в компьютерном классе. Это занимает около 5 минут на один компьютер.

## Шаг 2: Подключение Arduino

После установки Agent процесс подключения предельно прост:

1. **Подключите Arduino** к компьютеру через USB-кабель
2. Подождите **5-10 секунд** — компьютер распознает устройство
3. В CodeStudio нажмите кнопку **«Подключить устройство»** в верхней панели
4. В появившемся списке выберите **вашу Arduino** (обычно отображается как «Arduino Uno» или «USB Serial Device»)
5. Нажмите **«Подключить»**

Если всё прошло успешно, индикатор подключения станет **зелёным**, и вы увидите название вашей платы в верхнем правом углу экрана.

### Что делать, если Arduino не появляется в списке

В 90% случаев проблема решается одним из этих действий:

- **Переподключите USB-кабель** — иногда контакт неплотный
- **Попробуйте другой USB-порт** — некоторые порты могут быть заняты или неисправны
- **Проверьте, что Agent запущен** — иконка в системном трее должна быть активна
- **Перезагрузите страницу CodeStudio** — иногда помогает простое обновление
- **Попробуйте другой кабель** — дешёвые кабели для зарядки могут не передавать данные

## Шаг 3: Загрузка первой программы

Теперь проверим, что всё работает:

1. Создайте новый проект в CodeStudio, выбрав тип **«Arduino»**
2. В редакторе уже будет шаблон программы «Мигающий светодиод»
3. Нажмите кнопку **«Загрузить на плату»** (синяя кнопка с иконкой стрелки вниз)
4. Подождите 10-15 секунд — идёт компиляция и загрузка
5. Встроенный светодиод на плате Arduino начнёт мигать

**Поздравляем!** Вы только что загрузили первую программу на Arduino через браузер.

## Настройка для всего класса

### Подготовка компьютерного класса

Для урока с Arduino вам нужно подготовить каждое рабочее место:

1. **Установите Agent** на каждый компьютер (один раз)
2. **Подключите Arduino** к каждому компьютеру
3. **Создайте класс** в CodeStudio и добавьте учеников
4. **Назначьте задание** — ученики получат шаблон проекта автоматически

### Рекомендации по рассадке

- **Один Arduino на двух учеников** — оптимальный вариант для совместной работы
- Один ученик пишет код, второй собирает схему — затем меняются
- При нехватке оборудования можно организовать ротацию по станциям

### Мониторинг работы учеников

CodeStudio позволяет учителю видеть экраны всех учеников в реальном времени через панель управления классом. Вы можете:

- Видеть код каждого ученика
- Определять, кто застрял
- Отправлять подсказки конкретному ученику
- Демонстрировать экран ученика всему классу (с его разрешения)

## Поддерживаемые платы

CodeStudio поддерживает не только Arduino Uno. Полный список совместимых плат:

**Arduino:**
- Arduino Uno (рекомендуется для начинающих)
- Arduino Mega (для сложных проектов с большим количеством пинов)
- Arduino Nano (компактный вариант)

**ESP32:**
- ESP32 DevKit (WiFi + Bluetooth, для IoT-проектов)
- ESP32-S3 (продвинутый вариант с USB)

**Micro:bit:**
- BBC Micro:bit v1 и v2

Каждая плата автоматически распознаётся, и CodeStudio подстраивает интерфейс под её возможности.

## Частые вопросы учителей

### «Нужен ли интернет для работы с Arduino?»

Да, CodeStudio — браузерное IDE, и для его работы нужен интернет. Однако скорость не обязательно должна быть высокой — достаточно стабильного соединения со скоростью от 2 Мбит/с.

### «Можно ли использовать Chromebook?»

Да, CodeStudio отлично работает на Chromebook. Agent для ChromeOS доступен в виде расширения браузера.

### «Что если ученик случайно загрузит неправильную программу?»

Ничего страшного не произойдёт. Arduino спроектирована так, что неправильная программа не может повредить плату. Достаточно загрузить новую программу поверх старой.

### «Как подключить несколько плат к одному компьютеру?»

CodeStudio поддерживает подключение нескольких плат одновременно. Каждая плата отображается как отдельное устройство, и вы можете выбирать, на какую из них загружать программу.

## Заключение

Подключение Arduino к CodeStudio — процесс, который занимает не более 10 минут, даже если вы делаете это впервые. Установите Agent, подключите плату, загрузите первую программу — и вы готовы к уроку. Никаких драйверов, никакой командной строки, никакого стресса. Просто откройте браузер и начните учить детей программированию.
